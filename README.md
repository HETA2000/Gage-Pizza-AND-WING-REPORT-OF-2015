# Gage-Pizza-AND-WING-REPORT-OF-2015
This Power BI dashboard analyzes sales performance for Gage Pizza &amp; Wings, providing insights into revenue trends, product performance, and customer purchasing behavior. The goal of this project was to transform raw transactional data into actionable business insights to support data-driven decision-making.
Key Features & Insights

📊 Total Revenue, Orders, and Average Order Value (AOV)

🍕 Top-Selling Menu Items

📈 Monthly & Daily Sales Trends

🕒 Peak Sales Hours Analysis

🏬 Location/Branch Performance (if applicable)

📦 Category-wise Sales Breakdown (Pizza, Wings, Drinks, etc.)

Tools & Technologies

Power BI

DAX (Data Analysis Expressions)

Data Modeling

Power Query (ETL & Data Cleaning)

Business Impact
This dashboard helps management:

Identify best-performing products

Understand seasonal sales patterns
